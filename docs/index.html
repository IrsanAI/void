In einer Zeile installieren

Öffne Termux auf deinem Android-Gerät und führe diesen Befehl aus.

// Termux — einfügen & Enter drücken

$ curl -fsSL https://raw.githubusercontent.com/IrsanAI/void/master/install/install.sh | bash
// schritt für schritt

Termux installieren — kostenlos im F-Droid Store oder Google Play.

Termux öffnen und den kopierten Befehl einfügen. Mit Enter bestätigen.

Das Installer-Script läuft automatisch: Python & git werden bei Bedarf installiert, das Repo wird geklont, ein Alias void wird angelegt.

Nach der Installation direkt spielen — oder später mit void starten.

// nach der installation — einfach tippen

$ void

Der Installer richtet den Befehl void automatisch ein. Ab dem ersten Neustart von Termux reicht ein Wort.

// home-screen widget

VOID auf dem Home-Screen

Mit Termux:Widget kannst du VOID direkt vom Android Home-Screen starten — ohne Termux manuell zu öffnen.

Termux:Widget installieren — F-Droid (kostenlos, kein Account nötig).

Widget auf Home-Screen legen — Langer Druck auf den Home-Screen → Widgets → Termux:Widget → auf freie Fläche ziehen.

„VOID" auswählen — Der Installer hat den Shortcut automatisch angelegt. Einmal antippen → VOID startet sofort.

Der Shortcut-Script liegt automatisch in ~/.shortcuts/VOID und wird vom Installer erstellt. Termux:Widget liest genau diesen Ordner aus.

// psychologische mechaniken

Drei Emotionen. Ein Spiel.

VOID ist kein Zufall-Game. Jede Mechanik ist auf eine spezifische psychologische Wirkung ausgelegt.

Paranoia

Du weißt nie wo die VOID ist. Scans verraten sie kurz — aber enthüllen auch dich.

Soziale Bindung

Notsignal senden hilft dem Team — aber gibt deine Position preis. Vertrauen kostet.

Rivalität

Signale erhöhen deinen Score. Die VOID kommentiert, wer führt — und jagt den Besten.

Eskalation

Je länger du überlebst, desto aggressiver wird die VOID. Kein komfortables Endspiel.
