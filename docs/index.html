// installation

Wähle dein System

Öffne Termux auf deinem Android-Gerät.

// Termux — einfügen & Enter drücken

$ curl -fsSL https://raw.githubusercontent.com/IrsanAI/void/master/install/install.sh | bash

Termux installieren (F-Droid oder Play Store).

Befehl einfügen und mit Enter bestätigen.

Öffne a-Shell oder iSH auf deinem iPhone/iPad.

// iOS Terminal — einfügen & Enter drücken

$ curl -fsSL https://raw.githubusercontent.com/IrsanAI/void/master/install/install_ios.sh | sh

Hinweis: Der Installer erkennt iSH/a-Shell automatisch und installiert bei iSH die nötigen Basis-Pakete selbst.

a-Shell oder iSH aus dem App Store laden.

Befehl einfügen und mit Enter bestätigen.

Öffne PowerShell oder Windows Terminal.

// Windows — einfügen & Enter drücken

PS> irm https://raw.githubusercontent.com/IrsanAI/void/main/install/install_windows.ps1 | iex

Empfehlung: Windows Terminal + UTF-8 Font (z. B. Cascadia Mono), damit die Symbole korrekt aussehen. Windows-Installer installiert git/python nicht automatisch und prüft nur deren Verfügbarkeit.

Python und Git auf Windows installieren.

Befehl ausführen und im Launcher Solo oder Multiplayer wählen.

// nach der installation — einfach tippen

$ void

Der Befehl void startet das Spiel sofort.
